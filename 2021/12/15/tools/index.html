Transformers 库使用

安装

安装指导

基础条件：Python 3.6+, and PyTorch 1.1.0+ or TensorFlow 2.0+，基于conda创建一个虚拟环境，并进入虚拟环境，基于pip安装。（这里利用conda安装时导入库遇到了一些问题，pip安装没有问题，不知原因。）

pip install transformers

测试

python -c "from transformers import pipeline; print(pipeline('sentiment-analysis')('we love you'))"

首先会自动下载预训练模型，正常输出应为：

[{'label': 'POSITIVE', 'score': 0.9998704791069031}]
